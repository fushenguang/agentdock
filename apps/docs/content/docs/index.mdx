# AgentDock

AgentDock is an open-source scaffold platform for building production-ready AI coding agent projects.
It provides monorepo tooling, governance workflows, and opinionated templates so teams can move fast without drifting.

## Key Principles

- **Human owns intent** — AI operates within approved scope only.
- **OpenSpec = SSOT** — feature decisions live in `openspec/`, code reflects specs.
- **Anti-drift by design** — alignment checks and Layer 2 constraint gates catch drift before it ships.
- **pnpm + Turborepo** — fast, reproducible monorepo with strict TypeScript throughout.
