# AgentDock CLI

`@cogito.ai/cli` 是 AgentDock 平台的命令行工具，已正式发布并可用于生产。

它的核心设计理念：**同一套能力，同时服务于人类开发者、CI 流水线和 AI 编程 Agent**。

## 快速开始

```bash
# 交互式创建项目（推荐，面向人类开发者）
npx @cogito.ai/cli init

# 非交互式创建项目（面向 CI / Agent）
npx @cogito.ai/cli init --name my-app --template web-nextjs --pm pnpm --json

# 以 MCP Stdio 服务器方式启动（供 AI Agent 调用）
npx @cogito.ai/cli mcp
```

## 当前版本

| 版本 | 状态 | 说明 |
| ---- | ---- | ---- |
| 0.3.x | ✅ 正式可用 | `init`（交互/无头）+ `mcp`（Stdio 服务器） |

## 可用命令

| 命令 | 说明 |
| ---- | ---- |
| `agentdock init` | 从模板脚手架项目，支持交互/无头两种模式 |
| `agentdock mcp` | 启动 MCP Stdio 服务器，向 AI Agent 暴露工具接口 |
