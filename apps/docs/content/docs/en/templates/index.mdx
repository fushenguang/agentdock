# Templates

AgentDock provides production-ready project templates that follow the four-layer architecture and governance conventions.

## Available Templates

- **[web-nextjs](/en/docs/templates/web-nextjs)** — A Next.js 16 web application template with Supabase Auth, shadcn/ui, and i18n support.

## Template Philosophy

Each template is designed to be:

1. **Opinionated but flexible** — Sensible defaults with clear extension points
2. **AI-agent friendly** — Structured for Copilot, Cursor, and other AI coding assistants
3. **Production-ready** — Includes auth, validation, error handling, and testing setup
4. **Well-documented** — Clear architecture, conventions, and development workflow

## Using Templates

### Via CLI

```bash
npx @cogito.ai/cli create my-app --template web-nextjs
```

### Manual Clone

```bash
git clone https://github.com/cogito-tech/agentdock.git
cp -r agentdock/templates/web-nextjs my-app
cd my-app
pnpm install
```
