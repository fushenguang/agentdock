# AI Slop Anti-Patterns

## The 7 Forbidden Patterns

The following patterns are strictly forbidden in any UI generated by AI Coding Agents:

### 1. Inter + Roboto Font Dominance

**Description**: Using Inter or Roboto as the primary font for all text. These fonts are overused and lack brand identity.

**Detection**: Search for `font-family: Inter` or `font-family: Roboto` in generated CSS.

**Fix**: Use Geist, Plus Jakarta Sans, or another distinctive font as the primary.

### 2. Purple-to-Blue Gradients

**Description**: `from-purple-500 to-blue-500` gradients that scream "AI-generated."

**Detection**: Regex for `from-purple` + `to-blue` or similar gradient patterns.

**Fix**: Use `--color-primary` or solid colors from the design system.

### 3. Nested Rounded Cards

**Description**: Placing `rounded-xl` cards inside `rounded-xl` cards, creating a Russian doll effect.

**Detection**: Check for multiple nested elements with border-radius classes.

**Fix**: Use one level of border-radius. Inner elements should be flat or minimally rounded.

### 4. Pure Black / Pure Gray Text

**Description**: Text colors with no hue (e.g., `#000000`, `#808080`).

**Detection**: Check for exact black/gray hex values or `oklch(0 0 0)`.

**Fix**: Use neutrals with subtle hue (e.g., `oklch(0.52 0.012 262)` for muted text).

### 5. Bounce / Elastic Animations

**Description**: Animations with `bounce` or `elastic` easing that feel unprofessional.

**Detection**: Search for `bounce`, `elastic`, or spring animation configs.

**Fix**: Use `ease-out` or `cubic-bezier(0.4, 0, 0.2, 1)`.

### 6. Neon Glowing Header Text

**Description**: Text with `text-shadow` glow effects on dark backgrounds.

**Detection**: Search for `text-shadow` with glow/blur values on dark themes.

**Fix**: Remove glow. Use solid colors with proper contrast.

### 7. Icon Tile Above Every Heading

**Description**: Rounded square icon tiles (`rounded-lg bg-primary/10`) placed above every section heading.

**Detection**: Check for icon containers directly preceding heading elements.

**Fix**: Use icon + heading inline, or omit the icon entirely.

## Technical Quality Issues

Beyond the 7 forbidden patterns, also check for:

- **Contrast**: Text must have at least 4.5:1 contrast against its background
- **Touch targets**: All interactive elements must be at least 44x44px
- **Image optimization**: Images should use `next/image` with proper sizing
- **Horizontal scroll**: No horizontal scroll at any breakpoint

## Automated Detection

### Impeccable CLI

```bash
# Detect AI Slop in source code
npx impeccable detect src/ --fast

# Output JSON for CI integration
npx impeccable detect src/ --fast --json
```

The CLI uses deterministic rules (regex + AST analysis) to catch the 7 forbidden patterns without requiring an API key.

### CI Integration

```yaml
# .github/workflows/design-check.yml
- name: Detect AI Slop
  run: npx impeccable detect src/ --fast --json
```

**Note**: In MVP phase, do not fail CI on warnings. Only fail on confirmed AI Slop errors.

## Manual Review Checklist

During code review, check:

- [ ] Fonts are not Inter/Roboto as primary
- [ ] No purple-to-blue gradients
- [ ] No nested rounded cards
- [ ] Text colors have subtle hue
- [ ] No bounce/elastic animations
- [ ] No neon glow on text
- [ ] No icon tiles above every heading
- [ ] Touch targets >= 44px
- [ ] No horizontal scroll at 375px
