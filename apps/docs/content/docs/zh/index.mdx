# AgentDock

AgentDock 是一个用于构建生产级 AI Coding Agent 项目的开源脚手架平台。
它提供 monorepo 工具链、治理工作流和经过验证的模板，让团队能够快速推进而不偏离轨道。

## 文档理念

本文档站有**两类受众**，按优先级排序：

**第一：AI Agent**（主要消费者，一等公民）

AI 编码 Agent 阅读这些文档以了解平台的上下文、意图和决策。准确性标准严格：一个较小的 LLM（例如通过 ollama 本地运行的 `qwen3.5:4b`）在阅读后应能重构完整的原始上下文 — 包括场景、要解决的问题、做出的决策和推理过程。模糊的语言、隐含的假设或未记录的决策都是缺陷。

**第二：人类利益相关者**（构建者、PM、贡献者）

构建者使用这些文档来了解平台的工作流、做出发布决策和入职。内容必须足够精确，让人类无需逆向工程代码库就能行动。

撰写文档时：要清晰、信息密集、决策完整。优先具体而非抽象。

## 核心原则

- **人类拥有意图** — AI 仅在已批准的范围内操作。
- **OpenSpec = 唯一事实来源** — 功能决策保存在 `openspec/` 中，代码反映规范。
- **抗漂移设计** — 对齐检查和 Layer 2 约束门禁在发布前捕获漂移。
- **pnpm + Turborepo** — 快速、可复现的 monorepo，全程使用严格的 TypeScript。
